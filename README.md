## 🚀 Proyectos Destacados   

### 🚢 Modelo Predictivo de Consumo (Hackathon Balearia)
**Stack:** `Google Cloud Platform`, `Vertex AI`, `BigQuery`, `Python`
* **Arquitectura:** Diseño y ejecución de un pipeline de datos estructurado en GCP utilizando conjuntos de datos masivos proporcionados por la naviera.
* **Impacto:** Entrenamiento de modelos de Machine Learning para predecir y optimizar el consumo de combustible de la flota, aplicando técnicas de limpieza de datos (EDA) y preprocesado.

### 🤖 Alina: Ecosistema de Empleo con Agentes IA (Full-Stack)
**Stack:** `LangChain`, `React`, `Agentes IA`, `Bases de Datos`
* **Backend & IA:** Orquestación de agentes SDK secuenciales y en paralelo usando LangChain para generar simulaciones de entrevistas y alinear perfiles técnicos con ofertas de trabajo (LinkedIn).
* **Frontend:** Desarrollo de interfaz de usuario responsiva utilizando React, integrando los endpoints del modelo de IA con la capa visual.

### 🔄 Transformación de Datos en Data Warehouse 
**Stack:** `dbt (Data Build Tool)`, `SQL`, `Data Engineering`
* **Proyecto Práctico:** Orquestación y modelado de datos utilizando dbt-core (Open Source). Transformación de datos en crudo aplicando buenas prácticas de modularidad, versionado de código y tests de calidad en SQL.
